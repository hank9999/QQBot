# QQBot
基于NoneBot开发的QQ-MC机器人  
**部分功能需要安装插件 请谅解**  
**注意 Plugin开发并不规范 如需自行添加功能请查看NoneBot手册**  

## 快速开始
1. 按照[NoneBot指南](https://nonebot.cqp.moe/guide/)配置
2. 安装mcstatus (pip(3) install mcstatus)
3. 配置RCON `mcrcon_setting`
4. 配置超级管理员 `config.py`
5. 配置服务器信息 `status_setting.py`
6. python(3) main.py

## 指令
- #help
- #list or #status
- #mute
- #run
- #say
- #unmute
